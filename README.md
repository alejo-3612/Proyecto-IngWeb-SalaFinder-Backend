## SalaFinder:
Sala Finder es un programa web diseñado para la reserva de espacios dentro de la universidad. El sistema permite a los usuarios consultar la disponibilidad de espacios, realizar reservas y gestionar conflictos de horario. Dependiendo del tipo de espacio, las reservas pueden requerir aprobación por parte de un administrador. 

## Incluye:
Control de acceso por roles (Student, Staff y Admin), donde los administradores pueden gestionar espacios y aprobar o rechazar solicitudes.

Evita reservas en horarios superpuestos

Registra todos los cambios en un sistema de auditoría

Aplica políticas como el bloqueo de usuarios por inasistencias
